Атрибуты свойств

У нашего объекта есть 2 ключа — x и y. При этом с каждым из них связан набор атрибутов. Как, средствами JavaScript, получить сведения об объекте, подобные тем, которые показаны на предыдущем рисунке? Для этого можно использовать функцию Object.getOwnPropertyDescriptor(). Она принимает объект и имя его свойства, после чего возвращает объект, содержащий атрибуты этого свойства.
